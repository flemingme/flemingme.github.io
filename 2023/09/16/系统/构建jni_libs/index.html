构建jni_libs

  • 系统 App 本来就不该用 SDK 模式 —— SDK 模式是给第三方 App 或 SDK 库用的。
模式 特点 可见模块
平台模式（无 sdk_version） 用于构建系统镜像、系统 App 可见所有模块，包括 “libc++_shared”
SDK 模式（有 sdk_version） 用于构建面向第三方 App 的库 只能依赖 NDK/SDK 公共模块，如 “ndk_libc++_shared”
