git子模块创建提交与gerrit之间的细节

  1. 在本地创建好项目工程
  2. 拉取远程代码库
  3. 将本地导出的工程添加到克隆下来的文件夹中
  4. 将子模块add到项目中，并做好依赖关系
  5. 修改子模块的url地址

注意：gerrit权限的问题

  • 不能使用本地路径
  • 不能使用ssh://1016739@10.10.1.127:29418/app/android/library/radiolib这种个人链接
  • 要用相对地址../library/radiolib （一般android的项目都建立在app/android下）
    或者../../../../../app/android/library/radiolib
    注意是相对于根目录的所在地址，以免拉取不到对应的代码。
