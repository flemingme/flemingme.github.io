VS Code常用快捷键

VS Code 提供了丰富的快捷键，可以显著提高开发效率。以下是一些常用的快捷键分类整理（基于 Windows/Linux，macOS 用户可将 Ctrl 替换为 Cmd，部分快捷键可能因系统或设置不同而略有差异）：

基础操作

  • 打开命令面板：Ctrl + Shift + P（所有功能入口）
  • 新建文件/窗口：Ctrl + N / Ctrl + Shift + N
  • 保存文件：Ctrl + S
  • 保存所有文件：Ctrl + K → S
  • 关闭当前标签页：Ctrl + W
  • 撤销/重做：Ctrl + Z / Ctrl + Y

导航与搜索

  • 快速跳转文件：Ctrl + P（输入文件名）
  • 搜索文本：Ctrl + F（当前文件）
  • 全局搜索：Ctrl + Shift + F（跨文件搜索）
  • 跳转到定义：F12（或 Ctrl + 点击）
  • 返回/前进：Alt + ← / Alt + →（光标历史位置）
  • 跳转到行号：Ctrl + G

编辑代码

  • 复制/剪切当前行：Ctrl + C / Ctrl + X（无需选中）
  • 删除当前行：Ctrl + Shift + K
  • 上下移动行：Alt + ↑/↓
  • 代码格式化：Shift + Alt + F
  • 注释行：Ctrl + /（单行注释）
  • 多光标操作：
    • 添加光标：Alt + 点击
    • 批量选择相同词：Ctrl + D（连续按可选中多个）
    • 垂直多光标：Ctrl + Alt + ↑/↓

终端与面板

  • 切换终端：Ctrl + `` （反引号）
  • 新建终端：Ctrl + Shift + ``
  • 切换面板：Ctrl + Shift + E（资源管理器）、Ctrl + Shift + F（搜索）、Ctrl + Shift + D（调试）等
  • 隐藏侧边栏：Ctrl + B

调试相关

  • 启动调试：F5
  • 单步跳过：F10
  • 单步进入：F11
  • 切换断点：F9

窗口管理

  • 拆分编辑器：Ctrl + \
  • 切换标签页：Ctrl + Tab
  • 全屏模式：F11

自定义快捷键

  1. 打开快捷键设置：Ctrl + K → Ctrl + S
  2. 搜索命令并绑定新快捷键（如 Ctrl + M 绑定到某个功能）。

其他实用技巧

  • 重命名符号：F2（批量修改变量/函数名）
  • 折叠代码块：Ctrl + Shift + [ / ]
  • 快速修复：Ctrl + .（针对错误提示或建议）

提示：输入 Ctrl + K → Ctrl + R 可查看所有官方快捷键的参考手册（PDF）。
不同插件（如 Vim、ESLint）可能有额外快捷键，建议查阅插件文档。
